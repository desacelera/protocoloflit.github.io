# Corpo dos Sonhos – Landing Page

Landing page em HTML, CSS e JavaScript para promover o ebook "Corpo dos Sonhos" com foco em autocompaixão, nutrição estratégica e pacto de 30 dias.

## Visualizar

Abra o arquivo `index.html` diretamente em um navegador moderno ou, se preferir servir localmente, utilize qualquer servidor HTTP simples (por exemplo, Live Server no VS Code).

## Estrutura

- **Hero** com destaque para a chamada principal e botões de navegação suave.
- **Introdução e capítulos** descrevendo mentalidade, visualização criativa e nutrição estratégica.
- **Pacto de 30 dias** com animação cíclica destacando cada semana.
- **Bônus e CTA final** com formulário simulado.
- **Animações** suaves inspiradas na paleta do material de referência e efeito de revelação por rolagem.

## Tecnologias

- HTML semântico
- CSS com gradientes, animações e layout responsivo
- JavaScript para efeitos de scroll, destaque do pacto e validação simples do formulário
